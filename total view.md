# 🌍 Air Quality & Weather Data Analysis (Delhi)

## 📌 Project Overview

This project analyzes the relationship between air pollution (PM2.5 &
PM10) and weather conditions using a real-world dataset from Delhi
(2025).

## 📂 Dataset

-   348 daily records
-   Features: Temperature, Humidity, Wind Speed, Rainfall, Pressure,
    PM2.5, PM10

## 🛠️ Tools Used

-   Python
-   Pandas
-   NumPy
-   Matplotlib
-   Jupyter Notebook

## 🔧 Data Preprocessing

-   Checked missing values (none found)
-   Converted date column to datetime
-   Removed unnecessary columns
-   Checked duplicates (none found)

## 📊 Key Insights

-   Average PM2.5: 77.21
-   Max PM2.5: 240.82
-   Min PM2.5: 17.29
-   Pollution levels vary significantly over time
